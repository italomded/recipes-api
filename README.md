# Recipes API

Cooking recipes REST API. Created to practice learning.

## Docs
Docs are available on:
- `/api/docs` for JSON version;
- `/api/docs/ui` for Swagger-UI version.

## Main libs
- Spring Web;
- Spring Security;
- Spring Data;
- Lombok;
- OAuth0 JWT;
- JUnit;
- Mockito;
- Bean Validation;
- SpringDoc (OpenAPI 3).
